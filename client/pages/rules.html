Coding Quiz Rules

Read the rules carefully before participating in any coding quiz

1
General Rules

✓

Each quiz has a specific time limit. The timer starts as soon as you begin the quiz.

✓

Once you select an answer, you cannot go back to previous questions.

✓

All answers must be submitted before the timer runs out.

2
Scoring System

Correct Answer

+1 point for each correct answer

Incorrect Answer

No points deducted for wrong answers

3
Prohibited Actions

✗

Multiple accounts are strictly prohibited. Users caught with multiple accounts will be banned.

✗

Using automated scripts, bots, or any unfair means to answer questions is not allowed.

✗

Sharing quiz questions or answers during an active quiz session is prohibited.
